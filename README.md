## 🌟 Overview
Skin AI Care is an intelligent web application designed for **rural healthcare workers** to detect skin conditions using AI and provide **instant medical guidance**.

> 💡 *We focus on decision-making, not just prediction.*

---

## 🎯 Features

- 📸 **Image-Based Diagnosis**
- 🧠 **AI Prediction (CNN Model)**
- 📊 **Confidence Score**
- 🚦 **Severity Detection (Low / Medium / High)**
- 🚑 **Smart Referral System**
- 📖 **Simple Explanation**
- 🛡️ **Preventive Care Suggestions**
- 🌍 **Multi-Language Support (EN / हिंदी / ਪੰਜਾਬੀ)**
- 💾 **Save History (Database)**
- 📱 **Mobile-Friendly UI**
